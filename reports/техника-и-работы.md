# Техника и работы из Excel

> Источники: `Ведомость объемов работ трубы.xlsx`, `График реализации объекта.xlsx`, `Пример накопительной.xlsx`.

## Метод

- Сначала искались прямые упоминания строительной техники в строках Excel: например, `бульдозер`, `экскаватор`, `самосвал`, `каток`.
- Для неоднозначных слов добавлены ограничения: `кран` засчитывается только в контексте грузоподъемной техники, а не арматуры/сантехники; `каток` не засчитывается в выражении `катковые опоры`.
- Если техника явно не указана, она предполагалась только для строк, похожих на работы, а не на материалы: например, `уплотнение` -> каток/трамбовка, `разработка грунта` -> экскаватор/самосвал.
- Предположения надо проверить у профильного строителя: это не сметная норма, а первичная архитектурная выжимка для модели АКСГ.

## Как использовать дальше

- Использовать агрегированный список техники как черновик справочника `EquipmentType`.
- Связать типы техники с `WorkType`, а не с каждой строкой сметы один-в-один: в Excel много дублей и сметных уточнений.
- Для MVP АКСГ полезнее начать с типов техники, на которую реально можно поставить GNSS-модуль и камеру: самосвал, экскаватор, бульдозер, каток, кран, путевые машины.
- Для спорных работ оставить статус `требует проверки`, чтобы планировщик или инженер ПТО подтверждал технику вручную.
