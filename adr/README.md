# ADR

Architecture Decision Record фиксирует одно архитектурное решение, его контекст, альтернативы, последствия и условия пересмотра.

- [ADR-0003: Разделить локальный OCR и онлайн AI-функции](0003-разделить-локальный-ocr-и-онлайн-ai-функции.md)
- [ADR-0004: Отложить EAM-интеграцию и AR-очки](0004-отложить-eam-и-ar-очки.md)

## Статусы

- `Предложено` - решение обсуждается.
- `Принято` - решение действует в текущей архитектуре.
- `Заменено` - решение заменено другим ADR.
- `Отменено` - решение больше не применяется.
