# 13. Термины и сокращения

## Сокращения

| Сокращение | Расшифровка | Комментарий |
|---|---|---|
| АКСГ | Автоматизированный контрольно-сетевой график | Рабочее название платформы автоматизации КСГ и удаленного контроля стройки |
| КСГ | Контрольно-сетевой график | График работ, зависимостей, сроков, объемов и статусов строительства |
| ПСД | Проектно-сметная документация | Источник проектных решений, объемов, сметных данных и ограничений |
| СМР | Строительно-монтажные работы | Работы, выполнение которых контролируется в КСГ |
| ППР | Проект производства работ | Документ, описывающий технологию и порядок выполнения работ |
| ПОС | Проект организации строительства | Документ, задающий общую организацию строительства |
| BIM | Building Information Modeling | Информационная модель объекта; для MVP не является обязательным источником данных |
| GNSS | Global Navigation Satellite System | Общее название спутниковой навигации; в проекте рассматривается вместе с ГЛОНАСС/GPS |
| ГЛОНАСС | Глобальная навигационная спутниковая система | Навигация техники на объекте |
| БПЛА | Беспилотный летательный аппарат | Future scope для съемки и контроля объемов |
| IoT | Internet of Things | Датчики и устройства на технике/объекте |
| MVP | Minimum Viable Product | Первая версия продукта для проверки основной ценности |
| TBD | To Be Determined | Требует уточнения в следующих итерациях |

## Рабочие термины проекта

| Термин | Значение |
|---|---|
| Удаленный инспектор | Пользователь или подсистема, которая позволяет инспектору удаленно смотреть камеры на технике, проверять факт работ и вручную корректировать КСГ |
| Ручное ведение КСГ | Создание и изменение графика человеком через интерфейс, без обязательного автоматического разбора документов |
| Пикетаж | Линейная координата железнодорожного объекта, позволяющая привязать работы к участку пути |
| Факт работ | Подтверждение выполненной работы: вручную инспектором, через удаленный просмотр камер или через другой проверенный источник |
| Объективный сигнал | Данные GNSS/ГЛОНАСС, фото или видео с техники, которые помогают проверить работу, но сами по себе не закрывают работу в MVP |
